# 线性表
## 定义
- 有限的序列（有序的排列）
- 类型相同
- 可标记为(A1, ……,A(i-1),A(i),A(i+1),……,A(n))

## 抽象结构
- Data
- Operation
    - init
    - length
    - isEmpty
    - clear
    - get
    - locate(indexOf)
    - insert
    - delete
    - length

## 线性表的顺序存储结构
- 数据长度与线性表长度区别
数据长度是存放线性表存储空间的长度，存储分配后这个量一般是不变的（高级语言可以动态分配数组，不过带来了性能上的损耗）。

线性表长度是线性表数据元素的个数，这个量是变化的。任何时刻，线性表长度小于等于数据长度。

- 地址的计算方法
假设每个元素占用c个存储单元，那么第`i+1`个元素存储的位置和第i个元素的关系如下：
```
loc(A(i+1))=loc(A(i))+c
```
所以第i个元素与第一个元素的关系如下
```
loc(A(i))=loc(A(1))+(i-1)*c
```
通过这个公式，可以随时算出线性表中元素在内存中的地址。不管是第一个、最后一个或者其他元素，都是相同的获取时间，也就是一个常数。时间复杂度为O(1),我们通常把这一特点存储结构称之为随机存储结构。
